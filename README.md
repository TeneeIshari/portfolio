# Portfolio Website

This is my personal portfolio website built using **React** and **Tailwind CSS**. It showcases my projects, skills, and contact information in a modern and responsive design.

---

## Features

- **Responsive Design:** Works well on desktop, tablet, and mobile screens.
- **React Components:** Built using reusable React components for a modular structure.
- **Tailwind CSS Styling:** Modern and clean UI using Tailwind CSS utility classes.
- **Projects Section:** Displays projects with images and descriptions.
- **Contact Form:** Allows users to get in touch with me.
- **Smooth Animations:** Simple fade-in and hover animations for better user experience.

---

## Technologies Used

- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- HTML, CSS, JavaScript
- Vite (Development build tool)
